N O S H I

贈答（もらった・あげた）を AI が一緒に憶えておく。
撮るだけで、半返し・お返し・礼状まで。「贈答を、ちゃんと続けられる」。

MVP インタラクティブモック ▶ で物語が進みます

このモックの見かた

  • 下の黒いバーの ▶ で、利用者の体験が順番に進みます。
  • 画面内のボタンも実際に押せます（自由に行き来できます）。
  • 撮影 → 抽出 → 記録 → 半返し → お返し → 礼状、が中核の流れ。

デザイン意図

  • 生成り（washi）の地、墨の文字、水引の朱と金。
  • 表題は明朝、礼状は筆書体。贈答の“あらたまり”を UI に。
