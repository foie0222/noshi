# OWASP Lens — One-time Tailoring Answers (construction)

intent-001 で確定した暫定値を踏襲し、construction で具体化する。

- **扱うデータの機微度:** confidential 中心（贈答相手の氏名・関係・金額＝第三者PII）＋ 認証資格情報は restricted。
- **コンプライアンス:** 個人情報保護法（APPI）。決済導入時に PCI-DSS（今回スコープ外）。
- **認証モデル:** 外部 IdP（OAuth2/OIDC）＋メール。セッション/トークンに有効期限・失効。
- **公開範囲:** インターネット公開の消費者向け Web。
- **想定脅威:** A01（他人の贈答データ参照）、A03（フォーム/画像/プロンプト injection）、A07（認証悪用）、A09（監査）。
- **リスク許容度:** バランス型。個人情報漏えいは重大視。実装で本人スコープ強制・入力検証・暗号化・監査ログを作り込む。
