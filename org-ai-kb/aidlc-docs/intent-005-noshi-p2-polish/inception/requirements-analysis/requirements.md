# Requirements — noshi P2 polish（intent-005）

## Intent summary
- Type: feature（brownfield 磨き込み）/ Scope: 単一サービス noshi-service / Classification: brownfield / Affected repos: noshi
- 源泉: design-review 001 の P2。intent-001..004 の FR/NFR を保持。

## Functional requirements（新規・変更）

- **FR-5-1 ナビ再設計（P2-1）**
  - FR-5-1.1 撮影をタブから外し、中央の目立つ FAB（＋撮影）に格上げする。
  - FR-5-1.2 タブは ホーム / 台帳 / マイページ ＋ 中央の撮影 とする。

- **FR-5-2 水引モーション＋季節ナッジ（P2-2）**
  - FR-5-2.1 お返しを完了したとき、水引が結ばれる短いアニメーション（情緒的クロージャ）を表示する。
  - FR-5-2.2 季節（お中元: 6〜8月 / お歳暮: 11〜12月 / 年始: 12〜1月）に応じた優しいナッジをホームに控えめに表示する。

- **FR-5-3 オンボーディング・空状態（P2-3）**
  - FR-5-3.1 記録ゼロのホーム（空状態）は「まず1枚、撮ってみましょう」を中心に、撮影への明確な導線を出す。
  - FR-5-3.2 ログインの第一ボタンはやさしい言い回しにする（開発者語を排除）。

- **FR-5-4 アクセシビリティ（P2-4）**
  - FR-5-4.1 文字サイズ拡大トグル（標準/大）を提供し、設定が反映される。
  - FR-5-4.2 主要な画像/アイコン/ボタンに代替テキスト（aria-label 等）を付す。
  - FR-5-4.3 コントラストは WCAG AA を満たす（弔事トーン含む）。

- **FR-5-5 マイクロコピー（P2-5）**
  - FR-5-5.1 「はじめる(デモ)」等の開発者語・無機質な表現をやさしい日本語に置き換える。

- **FR-5-6 整理（負債）**
  - FR-5-6.1 未使用化した SummaryBar コンポーネント（と関連テスト）を削除する。

## Non-functional requirements
- NFR は intent-001..004 を維持。本人スコープ（A01）・入力検証・監査・分類不変。
- 季節ナッジ・文字サイズは確定的（日付・設定値）。アニメは prefers-reduced-motion を尊重する。
- パフォーマンス（初回表示 p95<2.5s）・モバイルファースト維持。

## Assumptions
- 技術スタック不変。視覚要素（モーション・レイアウト）は CSS 中心。
- 通知/リマインドは引き続きスコープ外。文字サイズ設定はローカル（端末内）保持で可。

## Out of scope
- ネイティブアプリ・音声入力（a11y の voice は将来）
- 全面リブランド・新規画面の大規模追加
- 季節ナッジの push 配信
