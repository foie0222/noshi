# Requirements Analysis — Clarification（intent-004 P1）

入力: design-review 001 / 既存実装。P1-1 トーン / P1-2 信頼可視化 / P1-3 贈与税110万枠。

### Q1: 贈与税の集計対象（P1-3 の核）
贈与税（暦年課税）は受贈者が1年間にもらった合計が110万円超で対象になりうる。ただし香典・お中元・お歳暮など社会通念上の贈答は通常非課税。

a) 香典・お中元・お歳暮を除外し、その他の「もらった」を暦年集計（より実態に近い）
b) もらった全額を暦年集計（単純・ただし過大に出る）

**Recommendation:** a) 社会通念上の贈答を除外。いずれも「税アドバイスではない・気づき」の免責を明記。

[Answer]: a) 香典・お中元・お歳暮を除外して暦年集計（免責付き）

### Q2: 贈与税の表示
a) 専用の小さなサマリ（今年もらった対象合計 / 110万まであと◯円 / 超過時は注意）＋免責文。マイページ的位置。
b) ホームに常時表示

**Recommendation:** a) 専用サマリ（控えめ）＋免責。常時前面化はしない。

[Answer]: a) 専用サマリ控えめ＋免責

### Q3: 弔事/慶事トーン（P1-1）
弔事の用途: 香典 / 御霊前 / 法事 / 弔慰 等。変更内容:
a) 配色を静かに（朱→墨/グレー）＋コピーを控えめに。モーションなし。
b) 配色のみ
c) フルにテーマ切替（アイコン・余白も）

**Recommendation:** a) 配色＋コピーの静かな切替（MVP）。

[Answer]: a) 配色＋コピーを静かに（モーションなし）

### Q4: 信頼の可視化（P1-2）
a) 相手（氏名）入力の近くに「あなただけが見られます🔒」、consent/設定にも同旨の安心文
b) consent のみ

**Recommendation:** a) 入力時＋consent/設定。felt trust を要所で。

[Answer]: a) 入力時＋consent/設定

### Q5: スコープ
[Answer]: P1-1/2/3 のみ。P2 は次 intent。stack 不変。贈与税は気づき（非税務助言）。
